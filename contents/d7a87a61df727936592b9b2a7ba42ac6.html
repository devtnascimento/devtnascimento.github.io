Gerenciar Perfil

UMLUseCase
Soy Stocks :: Casos de Usos :: Gerenciar Perfil

Description

Fluxo Principal - Acesso

1. O Usuario isere credenciais
2. O Sistema valida credenciais
3. O Usuario acessa o sistema

Fluxo excepcional - Bloqueio

1. Perfil existe mas credenciais foram inseridas incorretamente mais de 3 vezes
2. Sistema bloqueia o perfil
3. Sistema notifica usuario sobre bloqueio

Fluxo excepcional - Recuperacao

1. Perfil esta bloqueado
2. Usuario envia documentacao
3. Sistema valida documentacao
4. Usuario envia email para recuperacao de senha
5. Sistema identifica usuario
6. Usuario altera a senha
7. Sistema desbloqueia o perfil

Fluxo alternativo - Cadastro

1. Perfil nao existe
2. Usuario insere dados de cadastro
3. Usuario e redirecionado para o fluxo de acesso

Fluxo alternativo - Atualizacao

1. O Usuario insere novos dados  do perfil
2. Sistema identifica usuario
3. Sistema valida os dados
4. Sistema atualiza os dados
