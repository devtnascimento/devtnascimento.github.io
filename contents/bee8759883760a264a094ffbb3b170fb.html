Gerenciar Ordem

UMLUseCase
Soy Stocks :: Casos de Usos :: Gerenciar Ordem

Description

Fluxo Principal - Ordem a mercado

1. Usuario cadastra ordem de compra
2. Sistema Registra dados da ordem
3. Usuario seleciona opcao "Ordem a Mercado"
4. Sistema exibe Ordens com status de ativa
5. Ususario seleciona Ordens
6. Sistema altera status das Ordens selecionadas para "A mercado"

Fluxo Alternativo - Enviar Ordem

Processo Manual

1. Usuario seleciona Opcao "Enviar Ordem"
2. Sistema exibe Ordens ativas
3. Usuario seleciona Ordem a ser enviada
4. Sistema exibe opcoes de destino
5. Usuario seleciona opcao desejada
6. Sistema envia Ordem

Via Gatilho

1. Usuario seleciona opcao "Criar gatilho de envio"
2. Sistema exibe Ordens com status de ativa
3. Usuario seleciona Ordens a serem enviadas
4. Sistema exibe regras de envio (Data de envio, destino, meio de envio e formato do envio)
5. Usuario seleciona regras desejadas
6. Sistema registra gatilho de envio de Ordem
